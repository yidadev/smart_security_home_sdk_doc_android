获取传感设备列表

接口示例
void getSensorList(ITuyaResultCallback<Map<String, SensorBean>> callback);
参数说明
参数 说明
callback 获取结果回调
返回SensorBean参数说明
参数 类型 说明
deviceId String 设备id
actionInfos CameraStatusBean 绑定摄像头列表
返回CameraStatusBean参数说明
参数 类型 说明
entityId String 摄像头设备id
type int 摄像头绑定类型 1:绑定图片 2:绑定视频
示例说明
mTuyaSecuritySceneSdk.newEventRecordingInstance(房间id).getSensorList(new ITuyaResultCallback<Map<String, SensorBean>>() {
            @Override
            public void onSuccess(Map<String, SensorBean> result) {

            }

            @Override
            public void onError(String errorCode, String errorMessage) {

            }
        });

获取已绑定云存储的摄像头列表

接口说明
void getStorageBindCameraList(List<String> cameraIds, ITuyaResultCallback<ArrayList<String>> callback);
参数说明
参数 说明
cameraIds 所有摄像头id
callback 获取结果回调
回参说明

该接口返回已绑定云储存的摄像头列表 只有绑定了云储存的摄像头才支持联动视频

接口示例
mTuyaSecuritySceneSdk.newEventRecordingInstance(房间id).getStorageBindCameraList([所有摄像头的id], new ITuyaResultCallback<String>() {
            @Override
            public void onSuccess(String result) {

            }

            @Override
            public void onError(String errorCode, String errorMessage) {

            }
        });
