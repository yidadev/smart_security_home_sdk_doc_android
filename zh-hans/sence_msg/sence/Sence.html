安防场景

获取安防场景SDK

ITuyaSecuritySceneSdk mTuyaSecuritySceneSdk = TuyaOptimusSdk.getManager(ITuyaSecuritySceneSdk.class)

创建安防场景

接口说明
void createScene(String name, boolean stickyOnTop, String background, String displayColor, String coverIcon, List<SceneCondition> conditions, List<SceneTask> tasks, int matchType, final ITuyaResultCallback<SceneBean> callback);
参数说明
参数 说明
name 场景名称
stickyOnTop 是否出现在房间页头部
background 场景首页背景url
displayColor 场景图标颜色
coverIcon 场景前景图标
conditions 场景条件列表
tasks 场景动作列表
matchType 满足条件的类型，满足任意条件为1，满足所有条件为2
callback 获取结果回调
示例说明
mTuyaSecuritySceneSdk.newSceneInstance(家庭id).createScene("场景名称", "是否出现在首页",
                "背景图url", "图标颜色", "图标url", "条件列表", "动作列表", "满足类型",
                new ITuyaResultCallback<SceneBean>() {
                    @Override
                    public void onSuccess(SceneBean sceneBean) {

                    }

                    @Override
                    public void onError(String s, String s1) {

                    }
                });
