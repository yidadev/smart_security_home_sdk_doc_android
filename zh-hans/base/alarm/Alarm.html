获取布撤防SDK

ITuyaSecurityBaseSdk mTuyaSecurityBaseSdk = TuyaOptimusSdk.getManager(ITuyaSecurityBaseSdk.class)
HomeInfoBean 字段信息
字段 类型 描述
homeId String 家庭ID
mode int 家庭布撤防状态
alarmState int 家庭报警状态（0:正常；1:报警）
alarmVoice int 家庭报警声音状态（0:静音；1:报警）
alarmCountdown long 报警倒计时
AlarmActionBean 字段信息
字段 类型 描述
action String 报警操作动作描述
type int 报警操作（0:撤防；1:取消报警）
EmergencyContactBean 字段信息
字段 类型 描述
areaCode String 地区编号
phone String 电话号码
contractName String 联系人名称
MonitorServiceStateBean 字段信息
字段 类型 描述
state int 服务状态（0:Not Set；1:Self Monitoring;2:Pro Monitoring)
CountDownBean 字段信息
字段 类型 描述
deadline long 截止时间
currentMillis long 当前时间
AlarmMessageBean 字段信息
字段 类型 描述
alarmId String 报警ID
state int 消息状态
homeId String 家庭ID
msgId String 消息ID
msgContent String 消息内容
msgTitle String 消息标题
productType String 设备品类
productId String 设备品类ID
monitoringDeadline long MC中心截止时间
type String 报警类型
typeDesc String 报警类型描述
gmtCreate long 创建时间
deviceIds List 报警设备列表集合
